# Manu Loomba G — Astro Vaastu Consultant

A minimalistic, single-page website built with plain HTML, CSS, and vanilla
JS. No frameworks, no build step. Fully responsive and mobile-first.

## Files
- `index.html` — page markup (all sections)
- `style.css` — styling and responsive layout
- `script.js` — scroll reveal, sticky-header state, footer year

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server --directory site 8000
# then visit http://localhost:8000
```
